createPeerConnection()

Creates the peer connection and sets all needed event listeners (e.g. ICE candidates, negotiation, and data channels). The peer connection is created using the RTCPeerConnection API. The ICE servers are set to connect through a free Xirsys turn server. The data channel is created and event listeners are set for the data channel. The event listeners include: onicecandidate, onnegotiationneeded, ontrack, onopen, onmessage, and onerror.
